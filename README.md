# Melting
통합 유머 커뮤니티<br>
1인 개발 : 김지원<br>
개발 기간 :  2023년 05월 16일 ~ 2023년 07월 04일 (7주)<br><br>

## 📃 프로젝트 소개
Melting은 인기 유머 사이트를 통합한 커뮤니티 웹 사이트입니다.<br>
디시인사이드, 더쿠 등 다양한 인기 사이트를 실시간으로 제공하며 한곳에서 다양한 유머 사이트를 이용할 수 있습니다.
